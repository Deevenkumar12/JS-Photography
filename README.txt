================================================================
  HOW TO DEPLOY ONLINE (Make it live)
================================================================

OPTION 1 — NETLIFY (FREE, Easiest, Recommended)
  1. Go to: https://netlify.com
  2. Sign up with email (free)
  3. Click "Add new site" → "Deploy manually"
  4. Drag and drop the index.html file
  5. Done! Live URL example: https://js-photography.netlify.app
  6. Optional: Add custom domain (e.g., jsphotography.in)

OPTION 2 — GITHUB PAGES (FREE)
  1. Go to: https://github.com → Sign up
  2. Create new repository: "js-photography"
  3. Upload index.html
  4. Settings → Pages → Source: main
  5. Live at: https://yourusername.github.io/js-photography

OPTION 3 — HOSTINGER / GODADDY (Paid hosting)
  1. Buy hosting + domain on hostinger.in or godaddy.com
  2. Go to File Manager in cPanel
  3. Upload index.html to public_html folder
  4. Website is live at your domain!

OPTION 4 — ANY WEB HOST
  Just upload index.html via FTP or File Manager
  It works on any standard web server
